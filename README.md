# Quotes Book Web Scraping Project

## Project Description
This project is a web scraping application that extracts quotes from an online quotes book/website and stores them in a structured format for further analysis or use. It demonstrates the use of web scraping techniques using Python and popular libraries.

## Website scraped: http://quotes.toscrape.com/

## Technologies Used
Python
BeautifulSoup
Requests
Pandas (optional, for data storage)
CSV / JSON (for data output)

## Features
Scrapes quotes from a quotes book website
Extracts:
Quote text
Author name
Tags / Categories (if available)
Saves scraped data into CSV or JSON format
Easy to modify for other websites
